# Bree's Pizzeria

Local Vite + React + TypeScript project for a simple pizzeria site.

Getting started

1. Install dependencies

```bash
cd /home/bree/repositories/Mastering-React.Js/pizzeria
npm install
```

2. Run dev server

```bash
npm run dev
```

Pages: Home, About, Menu, Contact Us. Prices are displayed in Kenyan Shillings (KSh / KES). A dark mode toggle is available in the navbar.
